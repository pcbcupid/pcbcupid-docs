# GS006-T5848-1CH-MIC - I2S Digital Output Multi-Mode Microphone Module

## Overview

The GS006-T5848-1CH-MIC is a single-channel I2S digital output microphone module featuring the TDK InvenSense T5848 bottom-port multi-mode MEMS microphone. This module provides high-quality digital audio capture with I2S interface, making it ideal for voice recognition, audio recording, and acoustic sensing applications.

- **High-Performance Digital Microphone**: TDK InvenSense T5848 bottom-port MEMS microphone
- **I2S Digital Output**: Direct digital audio interface (SCK, WS, SD)
- **Multi-Mode Operation**: Configurable for different audio processing modes

## Hardware Description

### Microphone Core
The module is built around the TDK InvenSense T5848, a high-performance bottom-port MEMS microphone with I2S digital output. The T5848 features:
- Multi-mode operation (standard, low-power, and voice activity detection modes)
- Integrated wake-on-voice functionality
- Programmable threshold for voice activity detection
- Excellent acoustic performance with 65 dB SNR

### I2S Audio Interface
The module provides a standard I2S digital audio interface:
- **SCK (Bit Clock)**: Serial clock for synchronizing data transmission
- **WS (Word Select)**: Frame clock indicating left/right channel data
- **SD (Serial Data)**: Digital audio data output

### Power Supply
The module operates at 1.8V (VCC) as required by the T5848 microphone. The 3V3 prefix on some pins indicates 3.3V compatible logic levels for the I2S interface.

### Mechanical Design
- **Board Dimensions**: 25.4mm Ã 22.9mm
- **Mounting**: 4 mounting holes (2.54mm diameter) at corners for secure attachment
- **Connector**: 6-pin through-hole header (2.54mm pitch)
- **Acoustic Port**: Bottom-port design requires acoustic sealing on the PCB

## Getting Started

### Hardware Setup

1. **Mounting**: Secure the module using M2.5 screws through the four mounting holes
2. **Acoustic Sealing**: Ensure proper acoustic seal between the bottom port and PCB for optimal sound pickup
3. **Power**: Connect 1.8V to VCC (pin 1) and GND (pin 6)
4. **I2S Interface**: Connect SCK, WS, and SD to your microcontroller's I2S peripheral
5. **Channel Selection**: Configure LR pin for left/right channel assignment

### Software Configuration

1. **I2S Initialization**: Configure your microcontroller's I2S peripheral:
   - Master mode (module acts as slave)
   - Standard I2S format
   - Appropriate sample rate (8-48 kHz typical)
2. **Audio Data Processing**: Read audio samples from the I2S data line
3. **Wake-on-Voice**: Monitor WAKE pin for voice activity detection events
4. **Threshold Adjustment**: Use THSEL pin to adjust sensitivity for different environments
