## Functional Description

### I2S Digital Interface

The GS006 module uses the I2S (Inter-IC Sound) digital audio interface for communication. The I2S bus consists of three main signals:

- **SCK (Serial Clock)**: Bit clock for synchronizing data transmission
- **SD (Serial Data)**: Audio data output from the microphone
- **WS (Word Select)**: Channel selection signal (left/right)

### Multi-Mode Operation

The T5848 microphone supports multiple operating modes for optimizing power consumption versus performance:

- **Normal Mode**: High-performance audio capture
- **Low-Power Mode**: Reduced power consumption for always-on applications
- **Voice Activity Detection**: Wake-on-voice functionality with configurable thresholds

### Left/Right Channel Selection

The LR pin allows configuring the microphone for left or right channel operation in stereo I2S configurations. This enables multiple microphones to share the same I2S bus.
