## Prerequisites

Before installing Codex CLI, ensure the following are installed on your system:

* **Node.js** (includes `npm`)

  * Verify the installation:

    ```bash
    node -v
    npm -v
    ```

* A **ChatGPT** or **OpenAI API** account for authentication.

---

## Step 1: Install Codex CLI

Install the latest version of Codex CLI globally using npm:

```bash
npm install -g @openai/codex
```

![Installing Codex CLI NPM Command](https://files.pcbcupid.com/Documentation/Boards/prerequisites/codex/Codex_installation_check.avif)

---

## Step 2: Verify the Installation

Confirm that Codex CLI has been installed successfully:

```bash
codex --version
```

Example output:

```text
codex-cli x.y.z
```

---

## Step 3: Sign In

Authenticate using your ChatGPT or OpenAI account:

```bash
codex login
```

A browser window will open where you can complete the sign-in process.

![Authenticate Codex CLI](https://files.pcbcupid.com/Documentation/Boards/prerequisites/codex/Codex_login.avif)
<br/>
![Authenticate Codex CLI](https://files.pcbcupid.com/Documentation/Boards/prerequisites/codex/Codex_login_window.avif)

> **Note:** If you have previously signed in and encounter an authentication error (for example, an expired token), sign out first and then sign in again:

```bash
codex logout
codex login
```

---

## Step 4: Start Using Codex

Navigate to your project directory:

```bash
cd /path/to/your/project
```

Launch an interactive Codex session:

```bash
codex
```

You can also start Codex with an initial prompt:

```bash
codex "Explain this repository."
```

![Launch Codex CLI Session](https://files.pcbcupid.com/Documentation/Boards/prerequisites/codex/Codex_Launch.avif)

---

## Useful Commands

| Command           | Description                                                |
| ----------------- | ---------------------------------------------------------- |
| `codex`           | Start an interactive session.                              |
| `codex login`     | Sign in to your account.                                   |
| `codex logout`    | Remove stored authentication credentials.                  |
| `codex --version` | Display the installed version.                             |
| `codex --help`    | Show all available commands and options.                   |
| `codex doctor`    | Diagnose installation, authentication, and runtime issues. |
| `codex update`    | Update Codex CLI to the latest version.                    |

---

## Troubleshooting

### `codex` command not found

Verify that Codex CLI is installed globally:

```bash
npm list -g @openai/codex
```

If necessary, reinstall:

```bash
npm install -g @openai/codex
```

---

### Authentication failed

If you receive an authentication error such as an expired token:

```bash
codex logout
codex login
```

---

### Need Help

View the available commands:

```bash
codex --help
```

Run the built-in diagnostic tool:

```bash
codex doctor
```
