## Pin Configuration
 
- **VBUS** : +5V power from USB-C
- **CC1** : Configuration Channel 1 (used for cable orientation and power negotiation)
- **CC2** : Configuration Channel 2 (used for cable orientation and power negotiation)
- **D+** : USB 2.0 Data Positive
- **D-** : USB 2.0 Data Negative
- **GND** : Ground

## Key Features

- Breaks out USB-C pins to standard headers.
- Supports power and data lines.
- Compact and easy-to-integrate design.
- Ideal for rapid prototyping.

## Applications

- Custom PCB development
- USB-powered embedded systems
- Charging and power delivery projects
- Testing USB-C functionality

<Warning title="⚠️  Precaution Note">

- Ensure correct voltage and current ratings
- Verify CC pin configuration for proper power negotiation
- Avoid short circuits during prototyping
  
</Warning>
